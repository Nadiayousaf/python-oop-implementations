# python-oop-implementations
A collection of Python programs demonstrating core Object-Oriented Programming concepts such as classes, encapsulation, inheritance, polymorphism, and abstraction, along with real-world mini projects.
This repository contains my structured practice of Object-Oriented Programming (OOP) concepts in Python. It includes both fundamental implementations and real-world mini projects to strengthen practical understanding.

---

## 📌 Concepts Covered

- Classes and Objects  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Abstraction
